# Anti-Gravity Simulation Engine

A modern, web-based physics simulation engine featuring a responsive glassmorphic HUD dashboard control panel set against a deep cosmic dark-themed particle background. 

Particles spawn dynamically at the bottom boundary and accelerate upwards under simulated **Anti-Gravity**. Users can also interact with the viewport to create massive particle bursts or toggle a **Gravity Well (Vortex / Black Hole)** that pulls and swirls particles into a gravitational singularity.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/shanukhandala4-cell/anti-gravity-project)

## Features

- **Anti-Gravity Physics Core**: Standard HTML5 Canvas high-DPI (Retina) particle physics simulation.
- **Interactive Particle Settings**: Adjustable particle size, spawn density, upward lift acceleration, and horizontal wind/drift vectors.
- **Gravitational Manipulation (Gravity Well)**: Toggle a centripetal vortex black hole pulling and spiraling active particles towards the mouse/touch position.
- **Cosmic Themes**:
  - **Nebula Neon**: Bright pinks, purples, and cyan tones.
  - **Supernova Gold**: Fiery yellows, deep ambers, and burning red tones.
  - **Quantum Cyan**: Futuristic cyan, teals, and neon mint tones.
- **Responsive Layout**: Adapts dynamically from large high-DPI desktop viewports to mobile drawer-HUD interface layouts.
- **Self-Contained Deployment**: Entirely self-contained in a single `index.html` file containing HTML, inline styles, and pure JavaScript.
